# Sinar Logam - Landing Page

Landing page untuk usaha Sinar Logam menggunakan Next.js dan Tailwind CSS.

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser dan akses: `http://localhost:3000`

## Cara Build untuk Production

```bash
npm run build
npm start
```

## Fitur

- Hero section dengan branding Sinar Logam
- Section produk dengan kategori logam
- Keunggulan perusahaan
- Informasi tentang perusahaan
- Section kontak
- Responsive design untuk mobile dan desktop
- Desain modern dengan Tailwind CSS

## Customisasi

Anda dapat mengubah:
- Informasi kontak di section kontak (telepon, email, alamat)
- Warna tema di `tailwind.config.ts`
- Konten produk dan layanan di `app/page.tsx`
- Logo dan gambar sesuai kebutuhan

## Teknologi

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
